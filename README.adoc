= AsciiDoc Server - A SpringBoot application serving adoc pages as html

== Running
[source, bash]
----
./gradlew bootRun
----

== Configuration
When running asciidocserver it watches `classpath:/documentation` by default.
See: Link:./src/main/resources/application.yml
